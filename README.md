# Spanish Vocabulary Quiz

A simple and clean web application to help you learn basic Spanish vocabulary.

## Features

- **Vocabulary Quiz:** Test your knowledge of common Spanish words.
- **Instant Feedback:** See immediately if your answer was correct or incorrect.
- **Clean Interface:** A minimalist and user-friendly design.
- **Mobile-Friendly:** The app is responsive and works well on mobile browsers.

## How to Use

Simply open the `index.html` file in your web browser to start the quiz.
